README

How to install Marksale extention:

1. Copy id_access folder into typo3conf/ext/
2. Enable "Marksale" extention in configuration manager at backend 
(IMPORTANT typo3conf\PackageStates.php and typo3conf\LocalConfiguration.php should be writable)
3. Press configuration button is extention manager to enter key
4. Include static typoscript file of marksale extention into root template ( edit whole root template record and go to tab includes tab)
